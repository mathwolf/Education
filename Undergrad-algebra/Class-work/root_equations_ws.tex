\noindent {\sc {\bf {\Large Solving equations with square roots}}}
\bigskip
\bigskip

Review.  Find the vertex of the parabola.
\begin{enumerate}
	\item $y = x^2 - 3x + $ {\Large $\frac{1}{4}$}
\end{enumerate}

\bigskip

Solve the equation.

\begin{enumerate}[resume]    
    \item $\sqrt{4x + 1} = 3$
    \item $\sqrt{y-5} - 3 = 2$
    \item $\sqrt{x+1} + 1 = x$
    \item $\sqrt{x+3} - 1 = x$
    \item $\sqrt{x} - \sqrt{x-5} = 1$
\end{enumerate}

\bigskip

Determine the domain of each function and then graph it.

\begin{enumerate}[resume]
	\item $f(x) = \sqrt{x+2}$ \\
